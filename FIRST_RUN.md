ОБЩИЕ ПРОБЛЕМЫ И РЕШЕНИЯ:

❌ "Address already in use"
✅ Решение: Порт 25565 уже занят
   - Измените в include/globals.h: #define SERVER_PORT 25566
   - Пересоберите: make clean && make

❌ "gcc: command not found"
✅ Решение: Не установлен компилятор
   - Windows MSYS2: pacman -S mingw-w64-x86_64-gcc
   - Linux: sudo apt-get install build-essential
   - macOS: xcode-select --install

❌ Не видно мира / только чёрный экран
✅ Решение: Нормально, сервер грузит чанки. Подождите 5-10 сек

❌ Не могу подключиться (timeout)
✅ Решение: 
   - Убедитесь что сервер запущен
   - Убедитесь что адрес правильный (localhost:25565)
   - Проверьте firewall

❌ Сервер падает при подключении
✅ Решение: 
   - Запустите с дебагом: make debug-run
   - Посмотрите логи
   - Создайте issue с логами
